# Design System

Date: 2026-03-11

## Objective

VideoForge AI should feel like a premium product for performance teams: sharp hierarchy, dark cinematic energy, polished surfaces, and a dashboard that stays clear under data density.

## Visual Direction

- Marketing should feel aspirational, conversion-ready, and technologically credible.
- Dashboard surfaces should feel operational, not decorative.
- Motion should support clarity, not distract from generation and billing flows.

## Layout

- Mobile-first by default.
- Fixed header on mobile, desktop sidebar only when space allows.
- Marketing sections can breathe; dashboard sections should stay structured and dense.
- Mid-width desktop layouts must collapse earlier instead of compressing cards to unreadable widths.

## Surfaces

- Use layered dark surfaces with controlled contrast.
- Reserve high-contrast accents for CTA, status, and generation progress.
- Keep visual polish consistent between marketing and authenticated areas.

## Typography

- Headlines should feel decisive and modern.
- Dense dashboard labels should remain compact and readable.
- Pricing and CTA copy should stay short and forceful.

## Responsive Rules

- No horizontal overflow.
- Media, panels, and dialogs must fit within viewport width.
- Create Video, Billing, and History must keep action paths obvious on mobile.
